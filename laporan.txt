TUGAS RESTFUL API (LARAVEL) - Show, Update, dan Destroy

1. Fitur Show, Update, dan Destroy Data (Tabel Genre dan Author)
Menambahkan method `show($id)`, `update(Request $request, $id)`, dan `destroy($id)` pada `GenreController` dan `AuthorController`.
- `show` digunakan untuk mengambil dan menampilkan satu baris data spesifik berdasarkan ID.
- `update` digunakan untuk memvalidasi input dari user dan memperbarui data yang sudah ada di database.
- `destroy` digunakan untuk menghapus baris data di database berdasarkan ID.

2. Validasi Data Tidak Ditemukan (404 Not Found)
Mengatur validasi pada method `show`, `update`, dan `destroy`. Apabila data berdasarkan ID yang dikirimkan oleh klien tidak ditemukan di database, maka sistem akan mencegah terjadinya error aplikasi dengan mengembalikan response JSON berisi pesan `{"message": "Data tidak ditemukan"}` beserta kode status HTTP 404.

3. Mendaftarkan API Route menggunakan apiResource (routes/api.php)
Mengubah penulisan rute di `routes/api.php`. Rute yang sebelumnya menggunakan get dan post secara manual kini disederhanakan dengan menggunakan `Route::apiResource()`. Kode diubah menjadi:
Route::apiResource('genres', GenreController::class);
Route::apiResource('authors', AuthorController::class);
Route::apiResource('books', BookController::class);
Pendekatan ini akan mendaftarkan secara otomatis seluruh endpoint standar RESTful API (index, store, show, update, destroy).

4. Testing API Menggunakan POSTMAN
Melakukan pengujian fitur RESTful API yang baru dibuat:
- Request GET ke `/api/genres/{id}` dan `/api/authors/{id}` untuk mencoba fitur Show data.
- Request PUT (atau PATCH) ke `/api/genres/{id}` dan `/api/authors/{id}` menggunakan format raw JSON untuk mencoba fitur Update data.
- Request DELETE ke `/api/genres/{id}` dan `/api/authors/{id}` untuk mencoba fitur Destroy data.
- Menguji output 404 (Not Found) dengan mengirimkan parameter ID fiktif atau yang sudah dihapus sebelumnya. Pengecekan dilakukan untuk memastikan fitur respons data error berfungsi sesuai yang diharapkan.
