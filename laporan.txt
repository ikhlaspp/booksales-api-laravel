TUGAS MVC SEDERHANA & ELOQUENT DATABASE (LARAVEL)

1. Mengubah konfigurasi koneksi database (.env)
Menyesuaikan konfigurasi default Laravel agar terhubung dengan informasi database yang sudah ada (db_booksales).
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=db_booksales
DB_USERNAME=root
DB_PASSWORD=

2. Membuat file Model dengan Laravel Eloquent (app/Models/Genre.php)
Mendefinisikan class model (berlaku untuk Genre maupun Author) yang terhubung ke tabel. Menambahkan deklarasi `public $timestamps = false;` karena skema tabel tersebut tidak menggunakan format tipe datetime bawaan (created_at & updated_at).
<?php
namespace App\Models;
use Illuminate\Database\Eloquent\Model;

class Genre extends Model {
    public $timestamps = false;
    protected $fillable = ['name', 'description'];
}

3. Membuat Controller (app/Http/Controllers/GenreController.php)
Implementasi proses logika aplikasi. Controller memanggil operasi request `Genre::all()` untuk mengambil Collection Object Data MySQL dari Model, kemudian mengemasnya pada View tujuan beserta data payload array tersebut menggunakan fungsi `compact()`.
<?php
namespace App\Http\Controllers;
use App\Models\Genre;

class GenreController extends Controller {
    public function index() {
        $genres = Genre::all();
        return view('genres', compact('genres'));
    }
}

4. Mendaftarkan Route (routes/web.php)
Pengarahan endpoint atau rute aplikasi pada browser. Ketika pengguna mengakses '/genres', rute akan mengutus array class `[GenreController::class, 'index']` untuk diproses logika controllernya.
use App\Http\Controllers\GenreController;
Route::get('/genres', [GenreController::class, 'index']);

5. Menerapkan View dari Template Blade (resources/views/genres.blade.php)
Data dikodekan oleh Model menjadi format turunan Relasional (Bukan Associative Array tipe lama melainkan Eloquent/Object) sehingga referensi nilainya menggunakan syntax Object Pointer seperti `$genre->name`. Loop dinamis `@foreach` diterapkan untuk me-render HTML Table Body berdasarkan hasil Query dari Controller.
@extends('layouts.app')
@section('content')
    <h1>Daftar Genre</h1>
    <table border="1" cellpadding="10" cellspacing="0">
       <thead>
            <tr><th>ID</th><th>Nama Genre</th><th>Deskripsi</th></tr>
       </thead>
       <tbody>
           @foreach($genres as $genre)
           <tr>
               <td>{{ $genre->id }}</td>
               <td>{{ $genre->name }}</td>
               <td>{{ $genre->description }}</td>
           </tr>
           @endforeach
       </tbody>
    </table>
@endsection

Berikut tampilan pada halaman web ketika aplikasi sedang berjalan.
