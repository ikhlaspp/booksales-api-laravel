TUGAS RESTFUL API (LARAVEL)

1. Membuat Migration (Tabel books dan authors)
Menggunakan fitur Laravel Migration untuk membuat struktur tabel sesuai dengan susunan Brief yang tersedia. Pada tabel books ditambahkan foreign key yang merujuk pada tabel authors dan genres.

2. Membuat Data Dummy dengan Seeder (database/seeders/DatabaseSeeder.php)
Mengisi data dummy (masing-masing 10 data, memenuhi syarat minimal 5 data) untuk tabel authors dan books menggunakan class DatabaseSeeder.

3. Membuat Model dan Relasi Eloquent (app/Models/Book.php & Author.php)
Mendefinisikan class model untuk Author dan Book. Pada model Book ditambahkan relasi belongsTo ke Author dan Genre. Pada model Author ditambahkan relasi hasMany ke Book.
<?php
namespace App\Models;
use Illuminate\Database\Eloquent\Model;

class Book extends Model {
    public $timestamps = false;
    protected $fillable = ['title', 'description', 'price', 'stock', 'cover_photo', 'genre_id', 'author_id'];

    public function author() {
        return $this->belongsTo(Author::class);
    }
}

4. Mengatur Ulang Controller Menjadi API (app/Http/Controllers/BookController.php, dll)
Mengubah fungsi index() pada semua controller agar mengembalikan respons dalam format JSON memanfaatkan fungsi `response()->json()`.
<?php
namespace App\Http\Controllers;
use App\Models\Book;

class BookController extends Controller {
    public function index() {
        $books = Book::with(['author', 'genre'])->get();
        return response()->json($books);
    }
}

5. Mendaftarkan API Route (routes/api.php)
Semua endpoint URL sekarang dipindahkan dari file routes/web.php ke file routes/api.php agar dapat diakses dengan awalan endpoint '/api' (seperti /api/books).
use App\Http\Controllers\BookController;
use App\Http\Controllers\AuthorController;
use App\Http\Controllers\GenreController;

Route::get('/books', [BookController::class, 'index']);
Route::get('/authors', [AuthorController::class, 'index']);
Route::get('/genres', [GenreController::class, 'index']);

6. Testing API menggunakan POSTMAN
File view dari sistem template Blade (authors.blade.php, books.blade.php, dan genres.blade.php) telah dihapus dari direktori proyek karena pembelajaran saat ini ditujukan secara penuh untuk membangun API. Proses testing aplikasi kini menggunakan software API Client seperti POSTMAN dengan mengakses endpoint RESTful di atas.
